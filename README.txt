# User Management Web App

## Tech Stack
- Frontend: React.js
- Backend: Node.js + Express
- Database: MySQL

## Features
- User Registration & Login
- JWT Authentication
- Password Hashing with bcrypt
- User List with Edit/Delete
- Styled with Material-UI

## How to Run

### Backend
1. `cd backend`
2. `npm install`
3. Setup `.env` with DB credentials
4. `npm start`

### Frontend
1. `cd frontend`
2. `npm install`
3. `npm start`

## Database Setup
Run the SQL script in `db/schema.sql` to create the users table.

## Bonus
- JWT-based login sessions
- Material-UI for styling
